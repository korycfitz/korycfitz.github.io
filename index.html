Annonymous


GitHub Repository • Deployed Link

Annonymous, is a backend MEN stack social media platform, using Google Authentication.

This application is similar to the popular social media platform Yikyak, allowing users to share what's on their minds while staying completely anonymous. I decided to build this application because many social media users do not post what's on their minds. The fear of public perception often dissuades them from posting what they think. Yikyak, the most popular social media platform for anonymous posting, was removed in the United States. In response, I built this application to give back to the users their ability to stay anonymous while posting.

StyleSwipe


Front-End GitHub Repository • Back-End GitHub Repository • Deployed Link

StyleSwipe is a MERN stack application using JWT Authentication.

Upload, browse, and comment on outfits posted by other users across the web. StyleSwipe takes into account the logged-in user and can display all outfits of that particular user. We use passed parameters taken from the req.body to create a personalized index of all the logged-in user's outfits. This was a big stepping stone in understanding how information is passed from the back end to the front by URI calls.

PuppyCollector


GitHub Repository • Deployed Link

PuppyCollector is a PERN stack Django application, using AWS Authentication.

Collect puppies using this modern day Django reference application. PuppyCollector uses PostgreSQL, Express, React and Node.js in combination to allows users to create, update, delete, and read their "puppy cards". Furthermore,it restricts access to updating, deleting, and reading other user's "puppy cards", based on referencing the user through the "Puppy" model.
