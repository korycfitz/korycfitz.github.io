StyleSwipe


Front-End GitHub Repository • Back-End GitHub Repository • Deployed Link

StyleSwipe is a MERN-Stack application using JWT Authentication.

Upload, browse, and comment on outfits posted by other users across the web. StyleSwipe considers the logged-in user and can display all outfits of that particular user. We use passed parameters from the req.body to create a personalized index of all the logged-in user's outfits. Using passed-in parameters from req.body was a big stepping stone in understanding how URI calls passed information from the back end to the front.

PuppyCollector


GitHub Repository • Deployed Link

PuppyCollector is a PERN-Stack Django application using JWT Authentication.

Collect puppies using this modern day Django reference application. PuppyCollector uses PostgreSQL, Express, React, and Node to allow users to create, update, delete, and read their "puppy cards." Furthermore, it restricts access to updating, deleting, and reading other user's "puppy cards," based on referencing the user through the "Puppy" model.
